# Icons

Iconografía de BrewOS.

## Estilo oficial

- Outline (contorno), sin relleno
- Trazo 1.5px en escala 24px
- Esquinas redondeadas
- Librería base: **Lucide Icons**

## Iconos custom

Solo si Lucide no cubre una necesidad específica. Cualquier icono custom debe seguir las mismas reglas de trazo y tamaño.

## Estado

Sin iconos custom. Usar Lucide en implementación frontend.
