# Logos

Archivos de logo de BrewOS.

## Variantes previstas

Cuando se diseñe el logo final, incluir:

```
logos/
├── brewos-wordmark-light.svg
├── brewos-wordmark-dark.svg
├── brewos-isotype.svg
├── brewos-monogram.svg
├── brewos-full-light.svg      # isotipo + wordmark
├── brewos-full-dark.svg
└── favicon/
    ├── favicon.ico
    ├── favicon-16.png
    ├── favicon-32.png
    └── apple-touch-icon.png
```
